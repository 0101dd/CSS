01. Sass 介紹

Sass 是 CSS 預前處理器，目的在減少 CSS 的重工，以及更有效率維護樣式的結構，它須進行編譯 (complier) 成原生的 CSS，瀏覽器是看不懂 Sass。

Sass 可以使用變數、函式、流程控制、嵌套、繼承、混入等功能，被稱為 CSS 的程式語言(SassScript)。

02. Sass 的資料型態

  1. 1. number:數字有分有單位或無單位，1,12,133,10px,20rem。
  2. 2. string:有引號字串 'Noto Sans TC' 與無引號字串 sans-serif。
  3. 3. color:blue,#04a3f9,rgba(0,0,0,0.5),linear-gradient()。
  4. 4. boolean(布林值):true,false。
  5. 5. null(空值):null
  6. 6. list（像陣列）:用半形空格或逗號做為區隔，例如:1.5rem 1rem 0 2rem、10px,10px,20px,20px、Arial, Helvetica, sans-serif。
  7. 7. map (物件):類似 JavaScript 的物件。

03. 變數

。宣告使用 $，變數名稱的位置是它的作用域，就像 JS 的 let 宣告，在大括號裡面屬於大括號內的區域變數，寫在最外面屬於全域變數。

。變數的底線 _ 和連結線 - 可以互換使用，$bg_dark、$bg-dark 兩者是一樣的變數。

。!default 為預設，若該變數前後已經被賦值，會引用套用的值，!default 就不會被套用。

。使用變數可以維持網站一致性，節省修改成本。

。& 符號可以表示父層選擇器

06. Extend（繼承）%

。 繼承可以使用 CSS 選擇器 或 % 佔位選擇器來命名。

。 使用 CSS 選擇器當作繼承，會被編譯到 CSS，% 佔位選擇器屬於 SCSS 的範疇，不會被編譯到 CSS。

。 有共同樣式可使用繼承，會被聚集成群組選擇器，樣式碼較少。

。 @extend 沒有參數可以使用，mixins 有參數可以使用。

07. 計算

。 可以做加、減、乘、除、取餘數的計算。

。 Sass 的計算與 calc() 不同，Sass 預處理前已經計算好，不會造成瀏覽器的負擔。
