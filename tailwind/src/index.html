TailwindCSS

使用大量工具類 class，直接為結構撰寫樣式，具有彈性客製化的特性。

  • 不用再為元件去想 class 盟稱，直覺性寫樣式。
  • 開發者只需專注將 class 寫在 HTML 元素上。
  • 工具類 class 可以完全重複使用，很少去編寫新的樣式。

函式()與指令@

tailwind 提供函式與自訂屬於自己的 at-rule 指令讓我們使用。

@tailwind

使用 @tailwind 指令可以把 Tailwind 的 base、components、utilities 和 screens 樣式插入到 CSS 中。

@apply

提取 class 樣式來使用，可以多個 @apply

@layer

使用 @layer 指令來告知 Tailwind 這組自訂義的樣式屬於哪一個層級，例如 base、components、utilities。

@screen

@screen 指令可依照斷點名稱建立相關的媒體查詢。例如：

      @media (min-width: 640px){
        /* ... */
      }

使用 @screen 指令來產生

      @screen sm{
        /* ... */
      }

screen()

      @media screen(sm) {
        /* ... */
      }

theme()

使用 theme() 函式可引用 tailwind.confing.js 裡面的值，小數點要使用中括號[]。
